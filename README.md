## Project Overview
This project performs a comparative analysis of multiple machine learning algorithms for email spam classification.

We will:
- Preprocess email text data
- Apply TF-IDF feature engineering
- Train multiple supervised learning models (Naive Bayes, Logistic Regression, SVM)
- Evaluate performance using accuracy, precision, recall, and F1-score
- Deploy the best-performing model using Streamlit

## Project Structure
- preprocessing.py – Data cleaning and preparation
- feature_engineering.py – TF-IDF and vectorization
- models/ – Machine learning models
- evaluation.py – Model comparison and metrics
- app.py – Streamlit deployment
- main.py – Training pipeline
